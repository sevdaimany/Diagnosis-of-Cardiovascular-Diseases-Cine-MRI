

# Deep Learning-Based Diagnosis of Cardiovascular Diseases using 4D Cine MRI 

The aim of this research is to reveal the capabilities of deep learning in diagnosing cardiac
diseases and demonstrates the potential to generalize this technology to other medical fields.
